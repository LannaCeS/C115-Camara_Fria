# C115 - Câmara Fria
Código desenvolvido para o projeto de monitoramento de câmaras frias utilizando IoT.

## :pencil: Descrição
O presente projeto tem como objetivo projetar um sistema de controle de temperatura e umidade em câmaras frigoríficas voltadas para armazenamento de carnes em estabelecimentos alimentícios. O mesmo é composto por: dois tablets interconectados, um acoplado à parede interna e outro à externa da câmara; um sensor de temperatura e um sensor de umidade; e um site que dispõe as informações da câmara fria e emite avisos sobre eventuais situações críticas.

## :gear: Execução
1. Clonar o repositório
```bash
git clone https://github.com/LannaCeS/C115-Camara_Fria.git
cd C115-Camara_Fria
```
2. Envio de mensagem
```bash
cd camara_fria
python simulacao_leitura.py
```

3. Abrir o dashboard
O dashboard está presente no [Blynk](https://blynk.cloud).
